FONTS REQUIRED
==============
Copy "freepixel.ttf" from your original AlgoQuest LPC project into this folder.

Path expected by all scripts:
  res://assets/fonts/freepixel.ttf

Source: assets/codemon/font/freepixel.ttf  (from your algoquest_lpc project)

All scripts fall back gracefully if the font is missing — default Godot font
will be used. Copy it here for the proper pixel art look.
